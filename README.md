# Online Payment Fraud Detection System

This project uses Machine Learning to detect fraudulent online payment transactions.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Model
- Decision Tree Classifier

## Features
- Fraud transaction prediction
- Data visualization
- Confusion matrix
- Classification report
- Accuracy analysis

## Accuracy
Achieved approximately 99% accuracy.
